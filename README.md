# demo
人员信息管理管理系统，用户角色有管理员对学生和老师信息进行管理，实现群聊，实时刷新在线人数。
springboot + springMVC + mybatis +pagehelper + layui + beetle + websocket

(因为比较简单所以直接从mapper注入，懒得再写一层service了)
